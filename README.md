# Wish Wall Project Documentation
This project allows users to create wish lists and share them with others. Below are the key features of the project:

## Features
- Create a new wish list
- Add items to your wish list
- Share your wish list with friends

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/fengjianlin117/wish-wall2.git
   ```
2. Navigate into the project directory:
   ```bash
   cd wish-wall2
   ```
3. Install necessary packages:
   ```bash
   npm install
   ```

## Usage
To start the project, run:
```bash
npm start
```
